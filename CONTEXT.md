# 筷筷的博客

个人博客，用于记录技术、生活和阅读。访问者通过文章、标签和"此刻"动态了解博主近况。

## 核心概念

**文章 (Post)**:
一篇长文博客内容，以 Markdown 文件形式存放在 `src/pages/posts/`，包含标题、发布日期、标签、摘要等元数据。是博客的主要内容形式。
_Avoid_: 博文、帖子、笔记

**标签 (Tag)**:
附加在文章上的分类关键词，用于按主题筛选文章。每篇文章可有多个标签。
_Avoid_: 分类、类别、目录

**友链 (Friend Link)**:
指向其他博主或个人网站的外部链接，放置在 `/friends` 页面。表示博主之间的相互推荐。
_Avoid_: 友情链接、外链、推荐链接

**此刻 (Now)**:
受 nownownow.com 发起的 /now 页面运动启发，记录博主当下正在做什么、想什么、关注什么的短动态。内容来源于 Telegram 频道中带 `#now` 标签的消息，以倒叙时间线在 `/now` 页面展示。
_Avoid_: 动态、状态、瞬间、说说

## 辅助概念

**RSS**:
博客的 XML 订阅源，由 `/rss.xml` 端点生成，包含所有文章的内容摘要。仅包含文章，不含"此刻"动态。
